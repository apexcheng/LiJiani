# 电商运营助理简历主页

这是一个单页静态简历网站，适合部署到 GitHub Pages。

## 文件说明

- `index.html`：页面内容
- `style.css`：页面样式

## GitHub Pages 部署

1. 进入仓库 `Settings`
2. 打开 `Pages`
3. Source 选择 `Deploy from a branch`
4. Branch 选择 `main`
5. Folder 选择 `/root`
6. 保存后等待部署完成

最终访问地址一般是：

```text
https://apexcheng.github.io/LiJiani/
```
